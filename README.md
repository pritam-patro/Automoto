# Automoto

## Run Locally

Clone the project

```bash
  git clone https://github.com/pritam-patro/Automoto.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Tech Stack

HTML, CSS, Javascript
